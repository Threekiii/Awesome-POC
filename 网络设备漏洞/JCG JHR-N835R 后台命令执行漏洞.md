# JCG JHR-N835R 后台命令执行漏洞

## 漏洞描述

JCG JHR-N835R 后台存在命令执行，通过 ; 分割 ping 命令导致任意命令执行

## 漏洞影响

```
JCG JHR-N835R
```

## Shodan

```
JHR-N835R
```

## 漏洞复现

登录页面 admin admin登录
